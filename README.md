Welcome to my engineering and data science portfolio.  
This repository contains the source files for my GitHub Pages site, including project write‑ups, documentation, and supporting materials.

My work focuses on the intersection of **engineering, simulation, HVAC systems, and machine learning**. Many of these projects come from real operational challenges — systems that need to run smarter, safer, or more efficiently — and I use data, modeling, and control logic to push them forward.

---

## 🔧 What’s Inside

This repo includes Markdown pages for each project:

- **SIM‑1 Digital Twin** — A full‑physics HVAC simulation environment for testing real control logic  
- **Predictive Occupancy Modeling** — ML models predicting room occupancy from sensor data  
- **USDA Barley Market Analysis** — Large‑scale data cleaning and agricultural analytics  
- **Climate Change & Hop Production** — Statistical analysis of climate trends and crop yields  
- **Retail Sales Forecasting** — Time‑series forecasting using regression and Prophet  
- **RAG System with LangChain** — Retrieval‑augmented generation pipeline using ChromaDB  

All project pages are located in the root directory for GitHub Pages compatibility.
