# USDA Barley Market Analysis

This project focused on cleaning and analyzing a large USDA feed‑grains dataset (over 500,000 rows). The goal was to isolate U.S. barley data and explore pricing trends using both CSV data and web‑scraped information.

---

## What I Did

- Cleaned a 524,000‑row dataset  
- Removed unnecessary columns  
- Filtered for U.S. barley and barley‑malting data  
- Sorted and indexed the dataset for analysis  
- Scraped IndexMundi for domestic consumption data  
- Combined multiple data sources for trend evaluation  

---

## Tools Used

- Python (Pandas, Requests, BeautifulSoup)  
- CSV processing  
- Web scraping  
- Data filtering and transformation  

---

## Key Findings

- Extracted 586 relevant barley records  
- Identified pricing trends from 1978–2025  
- Successfully merged USDA and IndexMundi data  
- Built a clean dataset for future forecasting  

---

This project demonstrates large‑scale data cleaning, filtering, and integration — skills that translate directly to real‑world analytics work.
